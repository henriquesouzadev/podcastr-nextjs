## Getting Started

Node.js and npm

**Clone the project**

```bash
git clone https://github.com/henriquesouzadev/podcastr-nextjs.git
```

**Run the commmands**

```bash
# Install the dependencies
npm install

# Run the Fake API
npm run server

# Run the project in development mode
npm run dev
```

Access at `http://localhost:3000`
